O ObservaAção é um sistema, ainda na sua versão Beta, que permite a solicitação de reparos de bens públicos, de maneira simples. 

Ele oferece acesso a dois tipos de usuários:
* Cidadãos que desejam solicitar reparos;
* Servidores públicos que precisam gerenciar essas solicitações;

Algumas das funcionalidades são:
* Cidadão:
        - Criar uma solicitação;
        - Buscar alguma solicitação feita por ele;
        - Listar suas solicitações;
* Servidor público / Gestor:
       - Listar solicitações;
       - Atualizar status de uma solicitação;

Tecnologia:
- Java;

## Como Executar

1. Faça o download do projeto no formato **ZIP** e extraia os arquivos em uma pasta de sua preferência.
2. Execute o projeto a partir do arquivo `Main.java`.
3. O sistema será iniciado no **terminal**, que servirá como interface principal para uso da aplicação.
